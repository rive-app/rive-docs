![Toolbar](/images/interface/toolbar.png)

Use the toolbar to switch tools, create objects, adjust view options, invite collaborators, publish your file, and access file-level actions.

Common toolbar areas include:

- **Editor menu**: file settings, [revision history](/editor/fundamentals/revision-history), [export for runtime](/editor/exporting/exporting-for-runtime), [export for backup](/editor/exporting/exporting-for-backup), [share](/editor/embed-urls/overview), [rendering to video and image](/editor/exporting/exporting-for-video-and-static-design), and support links
- **Tools**: select, transform, [create artboards](/editor/fundamentals/artboards), [draw](/editor/fundamentals/shapes-and-paths-overview), [add rigging](/editor/manipulating-shapes/manipulating-shapes), and more
- **Invite**: Invite users to collaborate on this file
- **Publish**: [export](/editor/exporting/exporting-for-runtime), [embed](/editor/embed-urls/overview), publish to a [library](/editor/libraries), and publish to the [marketplace](/community/marketplace-overview#marketplace-overview)
- [**Data Bind Preview Toggle**](#data-binding-preview-toggle): preview bound data while editing

## Transform Tools menu

The Transform Tools menu houses tools that allow you to alter the transform properties of an object on the [Stage](/editor/interface-overview/stage).


![Select](/images/select.png)

**Select Tool**

The Select tool allows you to select objects and manipulate all their transform properties with the gizmo that appears. You can also click and drag in an empty space to select multiple objects with a marquee.

In most cases, you'll probably be using the Select tool, though we also currently offer a dedicated Translate, Scale and Rotation tools.

![Image](/images/editor/interface-overview/5cfc252a-8d52-4751-9064-4c6a1acb6f34.webp)

The gizmo allows you to manipulate the position, rotation, and scale properties of a selection.

**Translate Tool**

The Translate tool allows you to click on the gizmo or in an empty space to modify the position of your selected object without losing selection.


**Rotate Tool**

The Rotate tool allows you to click on the gizmo or in an empty space to modify the rotation of your selected object without losing selection.


**Scale Tool**


The Scale tool allows you to click on the gizmo or in an empty space to modify the Scale of your selected object without losing selection.

**Freeze Mode**

Freeze mode allows you to modify the position of origins, groups, and bones without affecting the underlying objects associated with them.

Learn more about Origin and Freeze mode [here](/editor/fundamentals/freeze-and-origin).
