Access design and animation tools from the Rive Editor toolbar. 

## Editor menu

The Editor menu is located to the left of the Transform Tools menu. This menu allows you to change the file name and view your file's [Revision History](../fundamentals/revision-history). Note that you can also change your file's name by double clicking on the tab at the top.

![Image](https://ucarecdn.com/de2dd753-07ef-46ac-a52e-1fc1ce371ee8/)

## Transform Tools menu

The Transform Tools menu houses tools that allow you to alter the transform properties of an object on the [Stage](./stage).

![Image](https://ucarecdn.com/6991eaed-093d-4bac-b7ac-d136d60defdf/)

The Select tool allows you to select objects and manipulate all their transform properties with the gizmo that appears. You can also click and drag in an empty space to select multiple objects with a marquee.

In most cases, you'll probably be using the Select tool, though we also currently offer a dedicated Translate tool. The Translate tool allows you to click in an empty space to modify the position of your selected object without losing selection. We plan to add dedicated Scale and Rotation tools as well.

![Image](https://ucarecdn.com/5cfc252a-8d52-4751-9064-4c6a1acb6f34/)

The gizmo allows you to manipulate the position, rotation, and scale properties of a selection.

## Export menu

Use the Export menu to export interactive content or traditional formats.

The Download option lets you download a runtime version of your file (.riv) to use with our[ ](https://app.outverse.com/rive/doc/getting-started/docxMZXGxnHR)[runtimes](https://app.outverse.com/rive/doc/getting-started/docxMZXGxnHR)[.](https://app.outverse.com/rive/doc/getting-started/docxMZXGxnHR) This is an optimized version of your file that strips out any data that is not necessary for runtime (such as unused assets or the position of states in state machine graphs). You can also download a backup of your file (.rev).

Use the [Share Link](https://app.outverse.com/rive/doc/overview/docCxp7bueYA)[ ](https://app.outverse.com/rive/doc/overview/docCxp7bueYA)option to share the current version of your file or easily embed it into another product without needing to use our runtimes.

Use the [Cloud Renderer](https://app.outverse.com/rive/doc/exporting/dockj1y5YeDd)[ ](https://app.outverse.com/rive/doc/exporting/dockj1y5YeDd)options to export a traditional non-interactive format like GIF, MP4, PNG, or WebM.

## Vector Tools menu

Design in Rive using the Pen Tool ([Pen Tool Overview](../fundamentals/pen-tool-overview)) and procedural shapes ([Procedural Shapes](../fundamentals/procedural-shapes)) . The Vector Tools menu houses all of the tools you'll need to create vector artwork.

![Image](https://ucarecdn.com/59977058-2afd-4ed7-8ab6-3057222d450e/)

## Bones menu

Access the Bone tool ([Bones](../manipulating-shapes/bones)) and Weight tool (when binding).

![Image](https://ucarecdn.com/a2406bf3-b2b7-49fd-960d-209d689309ab/)

## View Options menu

Customize the zoom level, enable snapping, and toggle the visibility of bones.

![Image](https://ucarecdn.com/ae5e6f53-5747-4040-bec3-254486ebe877/)

## Share Button

The Share button allows you to quickly create Share Links, publish a file to the Community, and download a .riv file.

![Image](https://ucarecdn.com/90822ce8-6df2-4443-aba5-ab525f1f4b0a/)

## Mode toggle

The mode toggle allows you to switch between Design and Animate mode. Press `Tab` to quickly switch modes.

![Image](https://ucarecdn.com/cc675849-3393-42fc-b3a3-38c93a7b7bd4/)
