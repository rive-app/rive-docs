## Stage

The stage is the main canvas where you create, arrange, select, and preview the contents of your file.

Artboards live on the stage. You can pan and zoom around the stage, select objects, and use guides or rulers to help position elements.

<Tip>
  To learn how to navigate and select items on the stage, see [Selection and Navigation](editor/interface-overview/selection-and-navigation).
</Tip>

### View Options menu

![View options](/images/interface/stage-view-options.png)

The View Options menu controls what appears on the stage while you work. Use it to adjust zoom, snapping, guides, and visual overlays for animation and interaction tools.

- **Zoom and snapping:** adjust the stage zoom level, enable snapping, or snap objects to pixels
- **User Cursors:** show or hide other users' cursors
- **Tool overlays:** show or hide gizmos, bones, targets, motion paths, joysticks, events, layouts, and text modifier ranges
- **Show Final Playback:** show the final playback state while editing
- **Rulers and guides:** show rulers, lock guides, or clear guides from the stag

## Toolbar

![Toolbar](/images/interface/toolbar.png)

Use the toolbar to switch tools, create objects, adjust view options, invite collaborators, publish your file, and access file-level actions.

Common toolbar areas include:

- **Editor menu**: file settings, [revision history](/editor/fundamentals/revision-history), [export for runtime](/editor/exporting/exporting-for-runtime), [export for backup](/editor/exporting/exporting-for-backup), [share](/editor/embed-urls/overview), and support links
- **Tools**: select, transform, [create artboards](/editor/fundamentals/artboards), [draw](/editor/fundamentals/shapes-and-paths-overview), [add rigging](/editor/manipulating-shapes/manipulating-shapes), and more
- **Invite**: Invite users to collaborate on this file
- **Publish**: [export](/editor/exporting/exporting-for-runtime), [embed](/editor/embed-urls/overview), publish to a [library](/editor/libraries), and publish to the [marketplace](/community/marketplace-overview#marketplace-overview)
- [**Data Bind Preview Toggle**](#data-binding-preview-toggle): preview bound data while editing

### Data Bind Preview Toggle

The Data Bind Preview Toggle lets you preview data binding values while editing, even when the state machine is not playing.

Use it when you want to see how bound values affect your file without entering playback or previewing the full state machine.

## Debug Panel

The debug panel shows messages, problems, file changes, testing tools, and audio tools while you work.

The debug panel includes these tabs:

- **Console**: view scripting and state machine console output
- **Problems**: view errors and warnings in your file
- **Changes**: review recent changes
- **Testing**: access testing tools for your file
- **Audio**: browse, preview, and clip audio assets

You can expand or hide the debug panel depending on how much space you need.
