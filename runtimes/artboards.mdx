## Selecting an artboard

You can specify which artboard to display when instantiating your Rive instance. If you don't specify an artboard, it will display the [default artboard](/editor/fundamentals/artboards#default-state-machine) as defined in the Rive Editor. If no default artboard is set, the first artboard in the file will be played.

    }

    ### SwiftUI

    ```swift
    class StateMachineViewController: UIViewController {
        var viewModel = RiveViewModel(
            fileName: "skills",
            stateMachineName: "Designer's Test",
            artboardName: "Square"
        )

        override public func loadView() {
            super.loadView()

            guard let stateMachineView = view as? StateMachineView else {
                fatalError("Could not find StateMachineView")
            }

            viewModel.setView(stateMachineView.riveView)
        }
    }
    ```
