---
title: "HTML Embed"
description: "Embed Rive in platforms that support custom HTML."
---

Use Rive in any platform that supports custom HTML embeds by pasting the Embed code generated from the Rive editor.

Some platforms also support embedding content directly from an Embed link.

This is useful for tools like **WordPress** and other platforms where you can add your own HTML, but a [dedicated Rive integration](/integrations/overview) or [runtime](/runtimes) is not available.

<Steps>
  <Step title="In the Rive editor, click the hamburger menu and select Generate Share Link">
    ![Generate share link](/images/integrations/html-embed/generate-share-link-1.png)

  <Step title="Click the Generate Link button">
    ![Generate share link](/images/integrations/html-embed/generate-share-link-2.png)
  </Step>
  <Step title="Copy the Embed code">
    Some platforms (like Notion) support embedding directly from an Embed link. Otherwise, use the iframe Embed code.

    <Info>
      For more information about the various types of share link, see [Share Links](/editor/share-links/overview#share-link-options).
    </Info>

    ![Generate share link](/images/integrations/html-embed/generate-share-link-3.png)
   </Step>
</Steps>

The embed code uses an iframe to display your Rive file.
