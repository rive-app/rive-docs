## Keyboard Shortcuts

| Command                          | macOS                                            | Windows/Linux                                     |
| -------------------------------- | ------------------------------------------------ | ------------------------------------------------- |
| **Search Scripts**            | <kbd>Cmd</kbd> + <kbd>P</kbd>                    | <kbd>Ctrl</kbd> + <kbd>P</kbd>                    |
| **Find Next**                    | <kbd>Cmd</kbd> + <kbd>G</kbd>                    | <kbd>Ctrl</kbd> + <kbd>G</kbd>                    |
| **Find Previous**                | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> |
| **Format File**    | <kbd>F4</kbd>                                    | <kbd>F4</kbd>                                     |
| **Toggle Comment** | <kbd>Cmd</kbd> + <kbd>/</kbd>                    | <kbd>Ctrl</kbd> + <kbd>/</kbd>                    |
| **Insert Line Below**    | <kbd>Cmd</kbd> + <kbd>Enter</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Enter</kbd>                    |
| **Insert Line Above**    | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> |
| **Outdent**              | <kbd>Shift</kbd> + <kbd>Tab</kbd>                    | <kbd>Shift</kbd> + <kbd>Tab</kbd>                     |
| **Move Line(s) Up/Down** | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>           | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>            |
| **Move Caret by Word**           | <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd>                   | <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd>  |
| **Move Caret by Subword**        | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | —                                           |
| **Move Caret to Line Boundary**  | <kbd>Cmd</kbd> + <kbd>←</kbd>/<kbd>→</kbd>                   | <kbd>Ctrl</kbd> + <kbd>←</kbd>/<kbd>→</kbd> |
| **Jump to Top/Bottom**           | <kbd>Cmd</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>                   | <kbd>Ctrl</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> |
| **Add Caret (multi-cursor)**     | <kbd>Alt</kbd> + Click                                       | <kbd>Alt</kbd> + Click                      |
| **Collapse to Single Caret**     | <kbd>Esc</kbd>                                               | <kbd>Esc</kbd>                              |
| **Accept Autocomplete**   | <kbd>Enter</kbd> or <kbd>Tab</kbd> | <kbd>Enter</kbd> or <kbd>Tab</kbd> |
