## Keyboard Shortcuts

### Palettes & Search

| Command                          | macOS                                            | Windows/Linux                                     | Notes                                      |
| -------------------------------- | ------------------------------------------------ | ------------------------------------------------- | ------------------------------------------ |
| **Open File Palette**            | <kbd>Cmd</kbd> + <kbd>P</kbd>                    | <kbd>Ctrl</kbd> + <kbd>P</kbd>                    | Opens the File Palette.                    |
| **Open Command Palette**         | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> | Opens the Command Palette.                 |
| **Search Workspace**             | <kbd>Cmd</kbd> + <kbd>K</kbd>                    | <kbd>Ctrl</kbd> + <kbd>K</kbd>                    | Opens the global search.                   |
| **Find**                         | <kbd>Cmd</kbd> + <kbd>F</kbd>                    | <kbd>Ctrl</kbd> + <kbd>F</kbd>                    | Opens the inline find panel.               |
| **Find Next**                    | <kbd>Cmd</kbd> + <kbd>G</kbd>                    | <kbd>Ctrl</kbd> + <kbd>G</kbd>                    | Also <kbd>Enter</kbd> inside the find box. |
| **Find Previous**                | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> | Moves to previous match.                   |
| **Close Find / Collapse Carets** | <kbd>Esc</kbd>                                   | <kbd>Esc</kbd>                                    | Closes find or collapses multiple carets.  |

### File & Edit

| Command            | macOS                                            | Windows/Linux                                     | Notes                           |
| ------------------ | ------------------------------------------------ | ------------------------------------------------- | ------------------------------- |
| **Save**           | <kbd>Cmd</kbd> + <kbd>S</kbd>                    | <kbd>Ctrl</kbd> + <kbd>S</kbd>                    | Saves current script.           |
| **Undo**           | <kbd>Cmd</kbd> + <kbd>Z</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Z</kbd>                    |                                 |
| **Redo**           | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> |                                 |
| **Copy**           | <kbd>Cmd</kbd> + <kbd>C</kbd>                    | <kbd>Ctrl</kbd> + <kbd>C</kbd>                    | Copies to the system clipboard. |
| **Cut**            | <kbd>Cmd</kbd> + <kbd>X</kbd>                    | <kbd>Ctrl</kbd> + <kbd>X</kbd>                    | Cuts selected text.             |
| **Paste**          | <kbd>Cmd</kbd> + <kbd>V</kbd>                    | <kbd>Ctrl</kbd> + <kbd>V</kbd>                    | Pastes from clipboard.          |
| **Select All**     | <kbd>Cmd</kbd> + <kbd>A</kbd>                    | <kbd>Ctrl</kbd> + <kbd>A</kbd>                    |                                 |
| **Format File**    | <kbd>F4</kbd>                                    | <kbd>F4</kbd>                                     | Formats and reindents the file. |
| **Toggle Comment** | <kbd>Cmd</kbd> + <kbd>/</kbd>                    | <kbd>Ctrl</kbd> + <kbd>/</kbd>                    | Comment or uncomment selection. |


➕ Inserting & Indenting

| Command                  | macOS                                                | Windows/Linux                                         | Notes                     |
| ------------------------ | ---------------------------------------------------- | ----------------------------------------------------- | ------------------------- |
| **Insert Line Below**    | <kbd>Cmd</kbd> + <kbd>Enter</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Enter</kbd>                    | Inserts a new line below. |
| **Insert Line Above**    | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> | Inserts a new line above. |
| **Indent**               | <kbd>Tab</kbd>                                       | <kbd>Tab</kbd>                                        | Indents selected lines.   |
| **Outdent**              | <kbd>Shift</kbd> + <kbd>Tab</kbd>                    | <kbd>Shift</kbd> + <kbd>Tab</kbd>                     | Decreases indentation.    |
| **Move Line(s) Up/Down** | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>           | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>            | Moves selected lines.     |


### Navigation & Selection

| Command                          | macOS                                                        | Windows/Linux                               | Notes                            |
| -------------------------------- | ------------------------------------------------------------ | ------------------------------------------- | -------------------------------- |
| **Move Caret Left/Right (char)** | <kbd>←</kbd> / <kbd>→</kbd>                                  | <kbd>←</kbd> / <kbd>→</kbd>                 | Hold <kbd>Shift</kbd> to select. |
| **Move Caret by Word**           | <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd>                   | <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd>  | Moves one word at a time.        |
| **Move Caret by Subword**        | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | —                                           | Only available on macOS.         |
| **Move Caret to Line Boundary**  | <kbd>Cmd</kbd> + <kbd>←</kbd>/<kbd>→</kbd>                   | <kbd>Ctrl</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | Jump to start or end of line.    |
| **Move Caret Up/Down**           | <kbd>↑</kbd> / <kbd>↓</kbd>                                  | <kbd>↑</kbd> / <kbd>↓</kbd>                 | Hold <kbd>Shift</kbd> to select. |
| **Jump to Top/Bottom**           | <kbd>Cmd</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>                   | <kbd>Ctrl</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> | Jump to file start/end.          |
| **Add Caret (multi-cursor)**     | <kbd>Alt</kbd> + Click                                       | <kbd>Alt</kbd> + Click                      | Adds another caret or selection. |
| **Extend Selection**             | <kbd>Shift</kbd> + Arrow/drag                                | <kbd>Shift</kbd> + Arrow/drag               | Extends from anchor to caret.    |
| **Collapse to Single Caret**     | <kbd>Esc</kbd>                                               | <kbd>Esc</kbd>                              | Cancels multiple selections.     |


### Autocomplete & Editing

| Command                   | macOS                              | Windows/Linux                      | Notes                        |
| ------------------------- | ---------------------------------- | ---------------------------------- | ---------------------------- |
| **Navigate Autocomplete** | <kbd>↑</kbd> / <kbd>↓</kbd>        | <kbd>↑</kbd> / <kbd>↓</kbd>        | Move through suggestions.    |
| **Accept Autocomplete**   | <kbd>Enter</kbd> or <kbd>Tab</kbd> | <kbd>Enter</kbd> or <kbd>Tab</kbd> | Inserts selected suggestion. |
| **Delete Forward**        | <kbd>Delete</kbd>                  | <kbd>Delete</kbd>                  | Deletes next character.      |
| **Delete Backward**       | <kbd>Backspace</kbd>               | <kbd>Backspace</kbd>               | Deletes previous character.  |
