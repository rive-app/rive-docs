## Keyboard Shortcuts

| Command                          | macOS                                            | Windows/Linux                                     |
| -------------------------------- | ------------------------------------------------ | ------------------------------------------------- |
| **Open File Palette**            | <kbd>Cmd</kbd> + <kbd>P</kbd>                    | <kbd>Ctrl</kbd> + <kbd>P</kbd>                    |
| **Open Command Palette**         | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd> |
| **Search Workspace**             | <kbd>Cmd</kbd> + <kbd>K</kbd>                    | <kbd>Ctrl</kbd> + <kbd>K</kbd>                    |
| **Find**                         | <kbd>Cmd</kbd> + <kbd>F</kbd>                    | <kbd>Ctrl</kbd> + <kbd>F</kbd>                    |
| **Find Next**                    | <kbd>Cmd</kbd> + <kbd>G</kbd>                    | <kbd>Ctrl</kbd> + <kbd>G</kbd>                    |
| **Find Previous**                | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> |
| **Close Find / Collapse Carets** | <kbd>Esc</kbd>                                   | <kbd>Esc</kbd>                                    |
| **Save**           | <kbd>Cmd</kbd> + <kbd>S</kbd>                    | <kbd>Ctrl</kbd> + <kbd>S</kbd>                    |
| **Undo**           | <kbd>Cmd</kbd> + <kbd>Z</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Z</kbd>                    |

| **Copy**           | <kbd>Cmd</kbd> + <kbd>C</kbd>                    | <kbd>Ctrl</kbd> + <kbd>C</kbd>                    |
| **Cut**            | <kbd>Cmd</kbd> + <kbd>X</kbd>                    | <kbd>Ctrl</kbd> + <kbd>X</kbd>                    |
| **Paste**          | <kbd>Cmd</kbd> + <kbd>V</kbd>                    | <kbd>Ctrl</kbd> + <kbd>V</kbd>                    |
| **Select All**     | <kbd>Cmd</kbd> + <kbd>A</kbd>                    | <kbd>Ctrl</kbd> + <kbd>A</kbd>                    |
| **Format File**    | <kbd>F4</kbd>                                    | <kbd>F4</kbd>                                     |

| **Insert Line Above**    | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> |
| **Indent**               | <kbd>Tab</kbd>                                       | <kbd>Tab</kbd>                                        |
| **Outdent**              | <kbd>Shift</kbd> + <kbd>Tab</kbd>                    | <kbd>Shift</kbd> + <kbd>Tab</kbd>                     |
| **Move Line(s) Up/Down** | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>           | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>            |
| **Move Caret Left/Right (char)** | <kbd>←</kbd> / <kbd>→</kbd>                                  | <kbd>←</kbd> / <kbd>→</kbd>                 |

| **Move Caret Up/Down**           | <kbd>↑</kbd> / <kbd>↓</kbd>                                  | <kbd>↑</kbd> / <kbd>↓</kbd>                 |
| **Jump to Top/Bottom**           | <kbd>Cmd</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd>                   | <kbd>Ctrl</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> |

| **Delete Forward**        | <kbd>Delete</kbd>                  | <kbd>Delete</kbd>                  |
| **Delete Backward**       | <kbd>Backspace</kbd>               | <kbd>Backspace</kbd>               |
