---
title: "Script Settings"
description: "Configure script optimization and debug information."
---

Script settings control how scripts are prepared for runtime. These settings apply at the file level.

To view script settings, deselect everything in the editor, then use the **Scripting** section in the Inspector.

## Optimization

**Optimization** controls how much Rive optimizes script code for runtime.

- **Full** - Applies the most optimization. Use this for production.
- **Medium** - Applies a balanced level of optimization.
- **None** - Skips optimization. Use this while debugging script behavior.

## Debug

**Debug** controls how much debugging information is included with scripts.

- **Full** - Includes the most debugging information. Use this while developing or troubleshooting scripts.
- **Medium** - Includes some debugging information.
- **None** - Removes debugging information. Use this for production when debugging information is not needed.

## Recommended Settings

For production, use higher optimization and lower debug settings.

For debugging, use lower optimization and higher debug settings.
