The Debug Panel lets you inspect script output and detect issues
in your code or data bindings.

Select the icon to toggle the debug panel open.

![Problems panel](/images/scripting/debugging/toggle-debug-panel.png)

The Debug Panel contains main tabs - Console and Problems.

## Console

The Console shows all log output from your scripts during playback. You can use the standard [Luau print()](https://create.roblox.com/docs/reference/engine/globals/LuaGlobals#print) function to log information, variable values, and messages.

```lua
print("Rive is so cool!")
print("Elapsed time:", seconds)
```

The Problems tab lists problems detected before the script runs â€” such as type mismatches, syntax errors, or missing data bindings.

The Problems tab handle shows the number of issues found in your scripts.
Click an item in the Problems panel to jump directly to the affected line of code.
Hover over any underlined code in the editor to see more details about the issue.
