There are two ways to create a new script.

**Creating a Script from the Assets Panel**

1. In the Data panel, click the `+` button.
2. Choose **Script** and select the [type of script](/scripting/script-types) you'd like to create.

TO DO: Image

**Creating a Script with the Scripts Tool**

1. Select the dropdown icon next to the Script button in the Toolbar
2. Select the [type of script](/scripting/script-types) you'd like to create.

When you create a new script, it will be added to the **Assets** panel and open in the Script Editor.
