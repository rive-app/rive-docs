## Methods

### `push`

<div class="signature">
```lua
push(vm: ViewModel) -> ()
```
</div>

### `insert`

<div class="signature">
```lua
insert(vm: ViewModel, index: number) -> ()
```
</div>

add view model at index value


### `swap`

<div class="signature">
```lua
swap(index1: number, index2: number) -> ()
```
</div>

swaps positions of view models at index1 and index2


### `pop`

<div class="signature">
```lua
pop() -> ViewModel?
```

### `shift`

<div class="signature">
```lua
shift() -> ViewModel?
```
</div>

### `clear`

<div class="signature">
```lua
clear() -> ()
```
</div>
