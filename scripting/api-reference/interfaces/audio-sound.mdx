---
title: AudioSound
---

Provides access to an AudioSound object


## Fields

### `volume`

volume of the sound.


## Methods

### `stop`

<div class="signature">
```lua
stop(fadeToStopTime: number?) -> ()
```
</div>

stop the audio.


### `seek`

<div class="signature">
```lua
seek(seconds: number) -> ()
```
</div>

seek the audio to a specific time in seconds.


### `seekFrame`

<div class="signature">
```lua
seekFrame(frame: number) -> ()
```
</div>

seek the audio to a specific time in PCM frames.


### `completed`

<div class="signature">
```lua
completed() -> boolean
```
</div>

Whether the sound has completed playing.


### `time`

<div class="signature">
```lua
time() -> number
```
</div>

Current sound time in seconds.


### `timeFrame`

<div class="signature">
```lua
timeFrame() -> number
```
</div>

Current sound time in PCM frames.


### `pause`

<div class="signature">
```lua
pause() -> ()
```
</div>

pauses the audio.


### `resume`

<div class="signature">
```lua
resume() -> ()
```
</div>

resumes the audio from its current position.


### `play`

<div class="signature">
```lua
play() -> ()
```
</div>

plays the audio.
