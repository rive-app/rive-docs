<Warning>
  The Rive Unreal plugin is currently in **technical preview**. The API may change before the official release.
</Warning>

This is the Unreal Engine runtime for [Rive](https://rive.app), allowing you to embed interactive Rive animations in your Unreal projects — in UMG widgets, on 3D surfaces, or anywhere you need dynamic, state-driven graphics.

For a complete list of supported features across all runtimes, see the [Feature Support](/docs/feature-support) page.

## Unreal Engine Version Support

The plugin supports Unreal Engine **5.7.3 and above**.

## Platform Support

| Platform | Status |
|----------|--------|
| Windows (64-bit) | Supported |
| macOS | Supported |
| Android | Coming soon |
| iOS | Coming soon |

<Info>
  Server targets are not supported — the plugin excludes server builds.
</Info>

## Rendering Support

| Backend | Platform |
|---------|----------|
| D3D11/D2D12 | Windows |
| Metal | macOS |

OpenGL (Android) and Metal (iOS) support is in development.

<Info>
  The Rive plugin uses RHI.
</Info>
